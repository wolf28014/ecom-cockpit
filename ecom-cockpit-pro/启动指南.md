# Windows 启动指南

## 🚀 三种启动方式（任选其一）

### 方式 1：双击 `run.bat`（推荐）

直接双击项目根目录的 `run.bat` 文件。

**如果双击后打开的是文本编辑器**（说明 `.bat` 关联被改了），请改用方式 2 或 3。

### 方式 2：右键 `run.ps1` → "使用 PowerShell 运行"

如果 PowerShell 提示"禁止运行脚本"，先执行：
```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```
然后再右键 `run.ps1` 运行。

### 方式 3：命令行启动（最稳定）

1. 按 `Win + R`，输入 `cmd`，回车打开命令提示符
2. 用 `cd` 进入项目目录，例如：
   ```
   cd /d D:\下载\ecom-cockpit-pro
   ```
3. 执行启动命令：
   ```
   run.bat
   ```
   或直接：
   ```
   python main.py
   ```

## 📦 打包成 EXE

如果要把应用打包成独立的 `.exe` 文件（不需要 Python 环境）：

### 方法 1：命令行执行（推荐）

```cmd
cd /d 项目目录
build.bat
```

### 方法 2：手动执行

```cmd
pip install -r requirements.txt
pip install pyinstaller
pyinstaller build.spec
```

打包完成后，在 `dist\电商经营驾驶舱Pro\` 目录下找到 `电商经营驾驶舱Pro.exe`，可直接双击运行。

## ❓ 常见问题

### Q1: 双击 `.bat` 文件打开了记事本/VSCode？

这说明 Windows 上 `.bat` 文件被关联到了文本编辑器。解决方法：

**临时方案**：右键 `.bat` 文件 → "打开方式" → "命令提示符"

**永久方案**：
1. 打开"设置" → "应用" → "默认应用"
2. 滚动到底部，点击"按文件类型选择默认应用"
3. 找到 `.bat`，设置为"命令提示符"或"Windows 命令处理程序"

### Q2: 提示"未检测到 Python"？

请到 https://www.python.org/downloads/ 下载 Python 3.10+，**安装时务必勾选 "Add Python to PATH"**。

### Q3: 依赖安装失败/很慢？

使用国内镜像加速：
```cmd
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q4: 启动后中文显示乱码？

在 cmd 中执行：
```cmd
chcp 65001
```
再运行 `python main.py`。

### Q5: PyInstaller 打包失败？

确保以下依赖已安装：
```cmd
pip install PySide6 PySide6-Fluent-Widgets PySide6-Addons pyinstaller
```

PySide6-Addons 提供 QtWebEngine（图表组件需要）。

启动后，应该看到：

2. 首页驾驶舱显示今日销售/利润/ROI 等 KPI 卡片

- 数据库文件位置：`%APPDATA%\EcomCockpitPro\ecom_cockpit.db`
- 备份目录：`%APPDATA%\EcomCockpitPro\backups\`
- 导出目录：`%APPDATA%\EcomCockpitPro\exports\`
